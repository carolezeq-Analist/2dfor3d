# 2dfor3d
Showcase of geospatial projects that transform 2D map data into 3D visualizations, using Blender
3D Reconstruction of a Historic Church – Jequié, Brazil
This project combines a Digital Elevation Model (DEM) of Jequié, Bahia,
with a conceptual 3D reconstruction of a historic church.
The objective is to explore terrain visualization, spatial context
and environmental representation using GIS and 3D tools.

Tools
- QGIS (DEM processing)
- Blender (3D modeling and rendering)

Applications
- Environmental analysis
- Urban and territorial visualization
- Cultural heritage visualization
